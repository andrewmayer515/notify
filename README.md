# Notify

## Setup
```
$ git clone https://github.com/andrewmayer515/notify.git
$ cd notify
$ npm i
```


Create .env at root with the following structure:
```
GMAIL_USER=example@domain.com
GMAIL_PASS=password
PING_DELAY=30
WEB_URL=https://www.something.com/
WEB_SELECTOR='.className'
```

Run with `npm start`
